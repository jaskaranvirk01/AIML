# Loan Approval Prediction using Machine Learning

This project predicts whether a loan will be approved based on applicant details.

## Steps Covered
- Data Cleaning & Imputation
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Encoding & Scaling
- Model Training & Evaluation

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes

## Libraries
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
